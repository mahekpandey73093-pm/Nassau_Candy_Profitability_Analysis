## 📌 Overview

This project presents an interactive Business Intelligence dashboard for analyzing product line profitability and gross margin performance for Nassau Candy Distributor. It helps identify high-performing products, low-margin products, division-wise profitability, cost-heavy products, and business improvement opportunities.

---

## 🎯 Objectives

- Analyze revenue, cost, and gross profit.
- Calculate gross margin and profit per unit.
- Compare division-wise financial performance.
- Perform Pareto (80/20) profitability analysis.
- Detect margin-risk products.
- Build an interactive Streamlit dashboard.
